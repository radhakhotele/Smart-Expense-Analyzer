# 💰 Smart Expense Analyzer

A personal finance dashboard built using Python, Pandas, Streamlit, and Plotly.

The application helps users track expenses, monitor budgets, analyze spending habits, and generate financial insights through an interactive dashboard.

---

## 🚀 Features

### Expense Management
- Add new expenses
- Categorize expenses
- Store expenses in CSV
- View recent expenses
- View complete expense history

### Data Import & Export
- Upload expense records using CSV
- Download expense data as CSV

### Analytics Dashboard
- Total spending overview
- Category-wise spending analysis
- Monthly spending trends
- Interactive Pie Chart
- Interactive Bar Chart

### Smart Financial Insights
- Highest spending category detection
- Largest transaction detection
- Weekend spending analysis
- Monthly spending comparison

### Budget Tracking
- Monthly budget setting
- Remaining budget calculation
- Budget utilization progress bar
- Overspending alerts
- Spending forecast

---

## 🛠️ Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
- CSV Storage

---

## 📊 Project Workflow

1. Add or upload expenses
2. Analyze spending patterns
3. Monitor budget usage
4. Receive financial insights
5. Forecast future spending

---

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
